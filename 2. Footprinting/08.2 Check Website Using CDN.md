#### 2Ô∏è‚É£ Check HTTP Response Headers

Use `curl` or browser developer tools to inspect HTTP headers.

```bash
curl -I https://example.com
```

Look for headers like:

-   `cf-ray`, `cf-cache-status` ‚Üí Cloudflare
-   `x-amz-cf-id` ‚Üí AWS CloudFront
-   `x-akamai-*`, `akamai-*` ‚Üí Akamai
-   `x-cache`, `x-served-by`, `via` ‚Üí Generic CDN or caching servers
